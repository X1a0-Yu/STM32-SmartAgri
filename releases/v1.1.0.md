# SmartAgri V1.1.0

**MCU：STM32F103C8T6**  
**MCU Protocol：2**

## 主要更新

- 新增 Flask 3.1 本地仪表盘与 REST/SSE 实时状态流。
- 通过 CH340 直接管理 MCU 串口，支持状态、阈值、执行器、LoRa 与 Wi-Fi 控制。
- 增加 SX1278 / Ra-02 LoRa 遥测路径。
- 增加 ESP-01S AT HTTP 上报实现。
- 完整串口遥测包含传感器数据、阈值、执行器状态、固件版本、Wi-Fi/LoRa 诊断信息。
- MCU 使用请求 ID 返回 `ACK` / `ERR`，支持安全的主机侧命令关联。

## 串口协议

- USART1：PA9/PA10，经 CH340 暴露，19200 8N1。
- MCU 上报：`DATA,...`、`ACK,<request_id>,...`、`ERR,<request_id>,...`。
- 主机命令：`GET`、`SET`、`CTRL`、`LORA`、`WIFI`。

## V1.2 迁移提醒

V1.1 的 Flask 进程会直接占用本地 CH340 串口，不适合放在远端服务器运行。V1.2 改为本地 Gateway 独占串口、MQTT 连接远端服务；V1.1 的 ESP HTTP 上传不作为 V1.2 的生产云链路。
