# SmartAgri V1.0.0

**发布日期：2026-07-14**  
**代号：V1.0 Baseline**

## 主要功能

- DHT11 温湿度采集。
- ADC 土壤、空气质量、光照采集。
- 水泵、风扇、补光灯继电器控制。
- 无源蜂鸣器告警、OLED 状态显示。
- 双按键阈值设置。
- USB/CH340 主机命令：`GET`、`SET`、`CTRL`。
- 带 TTL 的易失性执行器强制控制。

## 平台与烧录

- MCU：STM32F103C8T6。
- 主机接口：USART1 / PA9、PA10 / CH340，19200 8N1。
- Flash 地址：`0x08000000`。
- 固件：[SmartAgri_STM32_V1.0.0.hex](../烧录/V1.0.0/firmware/SmartAgri_STM32_V1.0.0.hex)。

## 已知限制

- 无源蜂鸣器以静态 GPIO 驱动时只会产生点击声，不能输出稳定音调。
- OLED 每 250ms 全屏清除后刷新，存在可见闪烁。
- 不包含 Ra-02 LoRa 或 ESP-01S Wi-Fi。
- Dashboard 为早期 FastAPI 路径，尚未切换到 Flask。
